# Care On

케어 서비스를 위한 Next.js 기반 웹 애플리케이션입니다.

## 기술 스택

- **Framework**: Next.js 14.2.16
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: ShadcnUI (Radix UI 기반)
- **Backend**: Supabase
- **Deployment**: Vercel

## 개발 환경 설정

### 1. 의존성 설치

\`\`\`bash
npm install
\`\`\`

### 2. 개발 서버 실행

\`\`\`bash
npm run dev
\`\`\`

개발 서버가 [http://localhost:3000](http://localhost:3000)에서 실행됩니다.

### 3. 빌드

\`\`\`bash
npm run build
\`\`\`

### 4. 프로덕션 서버 실행

\`\`\`bash
npm start
\`\`\`
